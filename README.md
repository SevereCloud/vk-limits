# VK лимиты

[Аналог лимитов для Telegram](http://limits.tginfo.me/ru)

## Команды

```sh
# Запускает live сервер
npm run start

# Запускает vk туннель
npm run tunnel

# Запускает форматирование
npm run format

# Запускает линтер
npm run linter

# Полностью собирает бандл для приложения
npm run build
```
